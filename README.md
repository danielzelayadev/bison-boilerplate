# Bison Boilerplate

A basic bison boilerplate.

![GNU](https://samskalicky.files.wordpress.com/2014/01/heckert_gnu_white.png)

## Getting Started

### Requirements

- [g++](https://gcc.gnu.org/)
- [make](https://www.gnu.org/software/make/)
- flex
- bison

### Build

```
make # or make build
```

### Run a Sample

Via shell script:

```
sh run.sh samples/sample.txt
```

Directly:

```
./bin/parser samples/sample.txt
```

### Clean

```
make clean
```
